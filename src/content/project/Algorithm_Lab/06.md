## Question1: Analyze the complexity of the above 0/1 Knapsack Algorithm.  
### Ans:  
1. Time Complexity:
* Outer loop runs n + 1 times (for each item $i = 0 -> n$)
* Inner loop runs W + 1 times (for each capacity $w = 0 -> W$)
* Therefore, the time complexity is: $O(n * W)$
2. Space Complexity:
* The memory used is: $(n + 1) * (W + 1)$
* Therefore, the space complexity is: $O(n * W)$

## Question2: Please explain why the above 0/1 Knapsack Algorithm is a pseudo polynomial time algorithm ...  
### Ans:  
The algorithm runs in $O(n × W)$ time. W is the numeric value of the knapsack capacity. The input size of W is only O(log W) bits. Since the runtime depends on the value of W, not its bit length, the algorithm is not polynomial in the input size. Therefore, it's a pseudo polynomial time algorithm, which is polynomial in the value of the inputs, but not in the size of the input in bits.

## Question3: Please refer to the above 0/1 Knapsack Algorithm and modify it to achieve a space complexity of $O(W)$.
### Ans:
```c++
int knapsack(int n, int W, int weights[], int values[]) {
    int dp[W + 1] = {0};
    for (int i = 0; i < n; i++) {
        for (int w = W; w >= weights[i]; w--) {
            dp[w] = max(dp[w], dp[w - weights[i]] + values[i]);
        }
    }
    return dp[W];
}
```

dp[i][w] =

dp[i - 1][w], & \text{if $w_i > w$} \\[2ex]
max(dp[i - 1][w], dp[i][w - w_i] + v_i), & \text{if $w_i \leq w$}
\end{cases}
$$

## Question5: Please modify the 0/1 Knapsack Algorithm into an Unbounded Knapsack Algorithm ...
### Ans:  
```c++
int knapsack_unbounded(int n, int W, int weights[], int values[], vector<int>& count) {
    vector<int> dp(W + 1, 0);
    vector<int> choice(W + 1, -1);
    for (int i = 0; i < n; i++) {
        for (int w = weights[i]; w <= W; w++) {
            if (dp[w] < dp[w - weights[i]] + values[i]) {
                dp[w] = dp[w - weights[i]] + values[i];
                choice[w] = i;
            }
        }
    }
    count.assign(n, 0);
    int w = W;
    while (w > 0 && choice[w] != -1) {
        int item = choice[w];
        count[item]++;
        w -= weights[item];
    }
    return dp[W];
}
```
## Question6: Please refer to the above Bounded Knapsack Algorithm and modify it using binary optimization ...  
### Ans:  
```c++
int boundedKnapsack(int n, int W, int weights[], int values[], int counts[]) {
    vector<int> dp(W + 1, 0); 
    for (int i = 0; i < n; ++i) {
        int w = weights[i];
        int v = values[i];
        int c = counts[i];
        for (int k = 1; c > 0; k <<= 1) {
            int use = min(k, c);
            int weight = use * w;
            int value = use * v;
            for (int j = W; j >= weight; --j) {
                dp[j] = max(dp[j], dp[j - weight] + value);
            }
            c -= use;
        }
    }
    return dp[W];
}
```
