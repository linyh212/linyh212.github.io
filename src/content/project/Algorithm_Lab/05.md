## Question1: Please add some constraints to let the result unique.  
### Ans:  
```c++
//Add a constructor to Node that stores the smallest character in the subtree.
//Update the node_comp comparator to use this character as a tie-breaker.
class HuffmanCode {
    struct Node {
        Node *e0, *e1;
        int freq;
        char ch;
        char min_char;
        Node(Node *e0, Node *e1, int freq, char ch) : e0(e0), e1(e1), freq(freq), ch(ch) {
            if (!e0 && !e1)
                min_char = ch;
            else
                min_char = std::min(e0->min_char, e1->min_char);
        }
    };

    Node *root;
    std::map<char, std::string> code;

    struct node_comp {
        bool operator()(const Node *a, const Node *b) const {
            if (a->freq != b->freq)
                return a->freq > b->freq;
            return a->min_char > b->min_char;
        }
    };

    void travel(const Node *v, const std::string &prefix) {
        if (!v->e0 && !v->e1) {
            code[v->ch] = prefix;
            return;
        }
        if (v->e0)
            travel(v->e0, prefix + "0");
        if (v->e1)
            travel(v->e1, prefix + "1");
    }

public:
    HuffmanCode(const std::string &text) {
        std::map<char, int> ch_count;
        for (char ch : text)
            ch_count[ch]++;

        std::priority_queue<Node*, std::vector<Node*>, node_comp> trees;

        for (const auto &p : ch_count)
            trees.push(new Node(nullptr, nullptr, p.second, p.first));

        while (true) {
            Node *t0 = trees.top(); trees.pop();
            if (trees.empty()) {
                root = t0;
                break;
            }
            Node *t1 = trees.top(); trees.pop();
            trees.push(new Node(t0, t1, t0->freq + t1->freq, -1));
        }

        travel(root, "");
    }

    std::string operator[](char ch) const {
        auto it = code.find(ch);
        if (it == code.end())
            throw std::runtime_error("Character not in alphabet");
        return it->second;
    }

    void print_codes() const {
        for (const auto &p : code)
            std::cout << p.first << ": " << p.second << '\n';
    }
};

```

## Question2: What is the time complexity of `HuffmanCode(text)`.  
### Ans:  
Assumptions:
n: Length of the input text
m: Number of distinct characters in the text
std::map: Each operation takes O(log m)
std::priority_queue: Each operation takes O(log m)
1. character frequencies
* Perform n insertions/lookups into the map
* Time complexity: O(n log m)
2. Inserting all characters into the priority queue
* Insert m nodes (one for each distinct character)
* Time complexity: O(m log m)
3. Constructing the Huffman Tree
* Combine nodes m - 1 times
* Each combination involves two removals and one insertion in the priority queue
* Time complexity: O(m log m)
4. Generating Huffman codes by traversing the tree
* The tree has about 2m - 1 nodes (binary tree)
* A full traversal takes linear time
* Time complexity: O(m)
5. Total Time Complexity: $O(nlogm+mlogm)$

## Question3: Please modify above algorithm to choose the minimum number ...

```c++
struct Activity {
    int s, e;
};

int min_cover(std::vector<Activity> A, int L, int R) {
    std::sort(A.begin(), A.end(), [](const Activity &a, const Activity &b) {
        return a.s < b.s;
    });
    int count = 0;
    int i = 0;
    int current = L;
    while (current < R) {
        int farthest = -1;
        while (i < A.size() && A[i].s <= current) {
            if (A[i].e > farthest) farthest = A[i].e;
            ++i;
        }
        if (farthest <= current) throw std::runtime_error("Cannot cover full range");
        current = farthest;
        ++count;
    }
    return count;
}
```

## Question4: Please design an algorithm to find how many rooms we need to hold ...
### Ans:
```c++
struct Activity {
    int s, e;
};

int min_rooms(std::vector<Activity> A) {
    std::vector<std::pair<int, int>> events;
    for (const auto &act : A) {
        events.emplace_back(act.s, +1);
        events.emplace_back(act.e, -1);
    }
    std::sort(events.begin(), events.end(), [](const auto &a, const auto &b) {
        if (a.first != b.first) return a.first < b.first;
        return a.second < b.second;
    });
    int max_rooms = 0;
    int current = 0;
    for (const auto &[time, type] : events) {
        current += type;
        max_rooms = std::max(max_rooms, current);
    }
    return max_rooms;
}
```

## Question5: Please discuss when we have multiple items with the same value/...  
### Ans:  
We should prioritize items with smaller weights, especially in the 0/1 knapsack problem. Choosing lighter items allows more flexibility in packing, as they leave more room for other items and reduce the chance of missing better combinations. While this tie breaking rule doesn't matter in the fractional knapsack since all fractions contribute equally, it can significantly affect results in the 0/1 version, where selecting a large item too early may block smaller, collectively more valuable items.
