## Problem 1-1:
fib(7)
= fib(6) + fib(5)
= (fib(5) + fib(4)) + (fib(4) + fib(3))
= ((fib(4) + fib(3)) + (fib(3) + fib(2)))
= (((fib(3) + fib(2)) + (fib(2) + fib(1))) + ((fib(2) + fib(1)) + (fib(1) + fib(0))))

**memo Table Updates:**
| Step | Function Call | Already Memoized? | memo Update |
|------|---------------|-------------------|-------------|
| 1    | fib(2)        | No                | memo[2] = 1 |
| 2    | fib(3)        | No                | memo[3] = 2 |
| 3    | fib(4)        | No                | memo[4] = 3 |
| 4    | fib(5)        | No                | memo[5] = 5 |
| 5    | fib(6)        | No                | memo[6] = 8 |
| 6    | fib(7)        | No                | memo[7] = 13|

Fibonacci number is stored in memo once it is computed. Future calls reuse the stored value.

---

| Method      | Time  | Space        |       |
|-------------|-------|--------------|-------|
| Naive       | O(2ⁿ) | O(n) (stack) | Exponential due to redundant calls |
| Tabulation  | O(n)  | O(n)         | Bottom-up; stores all values in a table |
| Memoization | O(n)  | O(n)         | Top-down with memoization; avoids recomputation recursively | 


## Problem 1-2:
```c++
#include <iostream>
#include <vector>

long long count_ways_to_climb(int n) {
    if(n < 0) return 0;
    std::vector<long long> dp(n + 1, 0);
    dp[0] = 1;
    for(int i = 1; i <= n; ++i) {
        if(i >= 1) dp[i] += dp[i - 1];
        if(i >= 2) dp[i] += dp[i - 2];
        if(i >= 3) dp[i] += dp[i - 3];
    }
    return dp[n];
}

int main() {
    using namespace std;
    cout << "--- Fibonacci / Climbing Stairs ---" << endl;    
    cout << "\n--- Climbing Stairs (1, 2, or 3 steps) ---" << endl;
    cout << "Ways to climb 0 stairs: " << count_ways_to_climb(0) << endl; // Expected: 1
    cout << "Ways to climb 1 stair: " << count_ways_to_climb(1) << endl;  // Expected: 1
    cout << "Ways to climb 2 stairs: " << count_ways_to_climb(2) << endl; // Expected: 2
    cout << "Ways to climb 3 stairs: " << count_ways_to_climb(3) << endl; // Expected: 4
    cout << "Ways to climb 4 stairs: " << count_ways_to_climb(4) << endl; // Expected: 7
    cout << "Ways to climb 5 stairs: " << count_ways_to_climb(5) << endl; // Expected: 13
    return 0;
}
```

## Problem 2-1:
Initialize dp = {1, 1, 1, 1, 1}

**Step 1. i = 1 (nums[1] = 10):**
Compare with j = 0 -> nums[0] = 3 < 10 -> dp[1] = max(1, dp[0]+1) = 2
dp = {1, 2, 1, 1, 1}

**Step 2. i = 2 (nums[2] = 2):**
Compare with j = 0 -> 3 > 2 -> skip
Compare with j = 1 -> 10 > 2 -> skip
No updates -> dp[2] remains 1
dp = {1, 2, 1, 1, 1}

**Step 3. i = 3 (nums[3] = 1):**
Compare with j = 0, 1, 2 -> all > 1 -> no updates
dp = {1, 2, 1, 1, 1}

**Step 4. i = 4 (nums[4] = 20):**
Compare with j = 0 -> 3 < 20 -> dp[4] = max(1, 1+1) = 2
j = 1 -> 10 < 20 -> dp[4] = max(2, 2+1) = 3
j = 2 -> 2 < 20 -> dp[4] = max(3, 1+1) = 3
j = 3 -> 1 < 20 -> dp[4] = max(3, 1+1) = 3
dp = {1, 2, 1, 1, 3}

**Step 5. LIS Length = 3**

---
Maintain an additional array prev where prev[i] stores the index of the previous element in the LIS ending at index i. While updating dp[i], if find that dp[j] + 1 > dp[i], then update both dp[i] and set prev[i] = j. After building the dp and prev arrays, find the index i where dp[i] is maximum, then backtrack using prev[i] to collect the elements of the LIS in reverse order. Finally, reverse this list to get the correct LIS sequence.
## Problem 2-2:
```c++
#include <iostream>
#include <vector>
#include <algorithm> // For std::max and std::max_element

int longest_increasing_subsequence(const std::vector<int>& nums) {
    if (nums.empty()) {
        return 0;
    }
    int n = nums.size();
    std::vector<int> dp(n, 1);
    for (int i = 1; i < n; ++i) {
        for (int j = 0; j < i; ++j) {
            if (nums[j] < nums[i]) { 
                dp[i] = std::max(dp[i], dp[j] + 1);
            }
        }
    }
    int max_length = *std::max_element(dp.begin(), dp.end());
    return max_length;
}

int main() {
    using namespace std;
    cout << "\n--- Longest Increasing Subsequence ---" << endl;
    vector<int> nums1 = {10, 9, 2, 5, 3, 7, 101, 18};
    cout << "LIS length for {10, 9, 2, 5, 3, 7, 101, 18}: "
         << longest_increasing_subsequence(nums1) << endl; // Expected: 4
    vector<int> nums2 = {0, 1, 0, 3, 2, 3};
    cout << "LIS length for {0, 1, 0, 3, 2, 3}: "
         << longest_increasing_subsequence(nums2) << endl; // Expected: 4
    vector<int> nums3 = {7, 7, 7, 7, 7};
    cout << "LIS length for {7, 7, 7, 7, 7}: "
         << longest_increasing_subsequence(nums3) << endl; // Expected: 1
    vector<int> nums4 = {3, 10, 2, 1, 20};
    cout << "LIS length for {3, 10, 2, 1, 20}: "
         << longest_increasing_subsequence(nums4) << endl; // Expected: 3
    vector<int> nums5 = {};
    cout << "LIS length for {}: "
         << longest_increasing_subsequence(nums5) << endl; // Expected: 0
    return 0;
}
```
