# Note #28 03/17/26  

# 會議主題： 人體偵測優化、3D 轉換問題與 UI 調整  

## 一、會議目的
- 更新人體偵測與 2D→3D 模型進度  
- 討論左右腳辨識問題對 3D 結果的影響  
- 確認 UI 調整方向與後續展示需求  
- 提醒與教授會議及專題展相關事項  

---

## 二、目前進度  
### 1. 人體偵測與 2D→3D 模型
- 已完成 **人體中心點偵測優化**  
  - 可正確鎖定主要人物  
  - 不再誤偵測背景或非目標人物  
- 已完成 **TCPFormer 2D → 3D 轉換模型**  
- 目前問題：  
  - 在 2D 訓練階段存在 **左右腳無法正確區分** 的情況  
  - 導致 3D 重建時出現錯誤（如姿態不合理或結構錯置）  

---

### 2. 問題分析與可能解法  
**問題來源：**
- 2D keypoints 本身左右標記不穩定  
- 錯誤會直接傳遞至 3D 模型（error propagation）

**可能解法：**
1. **重新訓練 2D 模型**
   - 強化左右腳標註與資料品質  
   - 增加資料多樣性（角度、遮擋）
2. **在 3D 階段加入限制條件**
   - 設定「腳不可越過身體中線」  
   - 作為後處理或約束條件

---

### 3. 網頁 UI 進度
