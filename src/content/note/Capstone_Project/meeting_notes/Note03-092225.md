# Note #03 09/22/25  

# 會議主題： 划船機姿勢分析系統進展  

## 一、會議目的  
- 以划船機為實驗平台，利用 8 台紅外線攝影機 + 身體標記點建立動作 2D → 3D 重建流程。  
- 討論側面攝影機 + YOLO 槳體偵測的進展與挑戰。  
- 明確下一步： 建立 ground truth、規劃實驗人數與次數。  

## 二、討論內容與決議  

### 1. 當前進度  
- 已針對範例影片進行 YOLO 初步測試，能大致框選出船槳位置，輸出為「框住船槳斜對角」的矩形方框。  
- 尚未結合紅外線攝影機與標記點資料，2D → 3D 人體動作重建仍在規劃中。  

### 2. 問題與挑戰  
- **背景與衣服顏色影響**：深色衣服或背景可能降低 YOLO 偵測準確度，導致方框不穩定。  
- **方框與船槳中心誤差**：目前 YOLO 輸出為對角方框，可能因槳角度或遮擋，造成方框中心與實際槳桿中心線偏移。  
- **數據一致性**：需思考如何校正 YOLO 偵測結果，使槳體中心軌跡更精準。  

### 3. 未來工作決議  
1. **建立 Ground Truth**  
   - 2D：由人工標記或額外追蹤方法建立槳體正確位置。  
   - 3D：將紅外線攝影機的標記點資料作為對照基準。  
2. **規劃實驗設計**  
   - 確定總共需要幾組單人數據集，包括訓練組 (training set) 與測試組 (test set)。  
   - 每位受試者需重複多少次划槳動作，才能形成足夠的樣本數以驗證模型正確性。  
3. **YOLO 偵測優化**  
   - 嘗試設定專門針對槳體的資料集（減少衣服、背景干擾）。  
   - 測試不同角度與光源，觀察模型對槳體位置偏移的敏感度。  
4. **槳體與骨架模型合併**  
   - 將 2D 槳體偵測與 2D → 3D 骨架模型結合，形成「持槳骨架模型」。  
   - 按照教練需求，分析划手與槳體的協調性、角度控制與節奏。  
5. **槳體軌跡細化**  
   - 槳體軌跡需進一步細分為多條直線片段（間隔可依據影格數或時間單位）。  
   - 逐段分析軌跡變化，用以檢視動作穩定度與一致性。  

## 三、未來修正與補充工作  
- 討論是否需要針對槳體增加標記點或顏色標示，協助 YOLO 偵測更穩定。  
- 製作可量化的指標（如：方框中心與實際槳中心線誤差距離）。  
- 建立「2D 偵測 vs 3D 重建」對照表，驗證 YOLO 偵測準確性。  
- 增加槳體軌跡分段分析，作為教練回饋的重要依據。  

## 四、後續思考  
- **資料量規劃**：多少受試者 + 每人多少組數據，才足以訓練與驗證模型？  
- **多源數據融合**：是否能結合 YOLO 偵測（影像）與紅外線標記點（3D）來互補？  
- **水上實驗延伸**：未來若移轉到水上場域，YOLO 偵測是否需重新訓練？  
- **教練端需求分析**：如何將槳體細化軌跡與骨架姿勢結合，轉換成教練可用的技術分析指標（如：槳入水角度、推進效率、一致性）。
